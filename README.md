## ✨ Overview

**AI Future World 2035** is a visually immersive, futuristic website built using pure HTML and CSS.  
It showcases:

- 🤖 AI Assistants  
- 🏠 Smart Living  
- 🚀 Future Transportation  
- 🌌 Neon Cyber Aesthetic
